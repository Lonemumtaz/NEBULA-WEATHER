# NEBULA WEATHER

Nebula Weather is a modern, responsive weather dashboard that allows users to search for real-time weather information for any city worldwide. It features a beautiful glassmorphism UI, dynamic backgrounds based on weather conditions, and supports both Celsius and Fahrenheit units.

## Features
- 🌦️ Real-time weather data using the OpenWeatherMap API
- 🔍 Search by city name (e.g., London, Tokyo, Paris)
- 🌡️ Toggle between Celsius and Fahrenheit
- 🎨 Dynamic backgrounds for different weather conditions
- 🚦 Loading, error, and success UI states
- 📱 Responsive design (works on desktop and mobile)

## Demo
Open `index.html` in your browser to try the app locally.

## Setup & Usage
1. **Clone or Download** this folder.
2. **Get an API Key:**
	 - Sign up at [OpenWeatherMap](https://openweathermap.org/) for a free API key.
	 - Replace the `API_KEY` value in `script.js` with your own key:
		 ```js
		 const API_KEY = "YOUR_API_KEY_HERE";
		 ```
3. **Open** `index.html` in your browser.
4. **Search** for any city and view the weather instantly!

## File Structure
- `index.html` — Main HTML file
- `script.js` — Handles API calls, UI updates, and logic
- `styles.css` — Custom styles (uses Tailwind CSS for core styling)
